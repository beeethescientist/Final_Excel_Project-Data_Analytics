# 📊 Excel Salary Dashboard  

![Salary Dashboard Preview](/0_Resources/Images/1_Salary_Dashboard_Final_Dashboard.gif)

## 📌 Introduction  
This project showcases an interactive Excel dashboard designed to explore salary trends across data-related roles globally. The goal was to transform raw job data into clear, visual insights that highlight how job title and location influence earning potential.

- 📊 Charts (Bar Chart, Map Chart)  
- 🧮 Formulas and Functions  
- ❎ Data Validation  
- 📋 Data Cleaning & Structuring

## 📊 Dataset  

The dataset includes job-related information for data roles across multiple countries. It contains:

- 👩🏾‍💻 Job Titles  
- 💰 Salary Data  
- 🌍 Locations  
- 🛠️ Skills  

This data was used to analyze salary patterns and compare global opportunities in the data field.

---

## 🧱 Dashboard Build  

### 📊 Data Science Job Salaries (Bar Chart)  
![Data_Science_Job_Salaries](/0_Resources/Images/1_Salary_Dashboard_Chart1.png)

- Used a horizontal bar chart to compare salaries across different job roles  
- Sorted salaries in descending order for easy comparison  
- Focused on making role-based salary differences clear  

💡 **Insight:**  
Higher-level and more technical roles generally offer higher salaries compared to analyst-level positions.

---

### 🗺️ Country Median Salaries (Map Chart)  
![Map_Chart Preview](/0_Resources/Images/1_Salary_Dashboard_Country_Map.gif)

- Used a map chart to visualize salary distribution by country  
- Applied color variation to highlight regional differences  
- Designed for quick understanding of global salary trends  

💡 **Insight:**  
Salary levels vary significantly by location, with certain regions offering higher compensation in the data industry.

---

### 🧮 Formulas and Functions  


- Applied formulas to calculate key metrics such as median salary  
- Used logical functions to filter and organize the data  
- Supported dynamic updates within the dashboard  

💡 **Purpose:**  
These calculations ensure that the dashboard reflects accurate and relevant insights based on selected criteria.

---

### ❎ Data Validation  
![Data_Validationn](/0_Resources/Images/1_Salary_Dashboard_Data_Validation.gif)

- Implemented dropdown filters for job title, country, and job type  
- Restricted user input to maintain consistency and accuracy  
- Improved overall usability of the dashboard  

💡 **Purpose:**  
Allows users to interact with the dashboard and explore different scenarios without affecting the data structure.
