# 📊 Global Data Analytics Jobs & Salary Analysis  

## 📌 Overview  
This project explores global trends in data-related careers, focusing on job roles, salaries, and required skills. 
Using Microsoft Excel, I analyzed job market data and built visual dashboards to better understand how the data industry varies across regions.

## 📁 Project Structure  
This repository is divided into two main parts:

- 📊 **Dashboard Project**  
  Interactive Excel dashboard visualizing salary trends and job distribution  
  👉 See more: `/dashboard/README.md`

- 📈 **Data Analysis Project**  
  In-depth analysis of skills, salaries, and job market trends  
  👉 See more: `/analysis/README.md`

## 🛠️ Tools Used  
- Microsoft Excel  
- Pivot Tables & Charts  
- Power Query  
- Power Pivot  
- Advanced Excel formulas  

## 🎯 Key Focus  
- Salary trends across different countries  
- Comparison of data roles (Data Analyst, Data Engineer, etc.)  
- Relationship between skills and salary  

## 🚀 About This Project  
This project is part of my journey transitioning into the data analytics field. 
It reflects my ability to work with real-world data, build dashboards, and extract meaningful insights.
