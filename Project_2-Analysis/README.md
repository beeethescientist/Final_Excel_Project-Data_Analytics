# 📈 Data Analysis: Global Data Job Market

## 📌 Overview  
This project explores real-world data science job postings to understand how skills, job roles, and geography influence salary levels in the data industry.

The analysis focuses on identifying patterns in job requirements and compensation to better understand what drives higher-paying roles.

---

## ❓ Key Questions Explored  

This analysis was guided by the following questions:

- Does having more skills lead to higher salaries?  
- How do salaries differ across countries and regions?  
- What skills are most commonly required in data roles?  
- Which skills are linked to the highest-paying jobs?  
- How does job title impact earning potential?  

---

## 🛠️ Tools & Excel Techniques Used  

- 📊 Pivot Tables for data summarisation and grouping  
- 📈 Pivot Charts for visual exploration of trends  
- 🔍 Power Query for data cleaning and transformation  
- 💪 Power Pivot for building relationships between datasets  
- 🧮 Advanced Excel formulas and calculated measures  
- 📉 Comparative analysis across roles and regions  
- 🧠 Exploratory Data Analysis (EDA)  
- 🧩 Pattern identification in structured job market data

## 📊 Analysis & Insights  

### 🧠 Skills vs Salary  
![Skills_vs_salary](0_Resources/Images/2_Project_Analysis_Chart1.png)
- Roles requiring a broader or more technical skill set tend to offer higher salaries  
- Senior and engineering-focused roles generally outperform generalist analyst roles in pay  
- There is a clear relationship between specialization and earning potential  

---

### 🌍 Regional Salary Differences  
![Regional_salary_differences](0_Resources/Images/2_Project_Analysis_Chart2.png)
- Salary levels vary significantly across countries  
- High-demand tech regions tend to offer higher compensation for similar roles  
- Location remains a key factor in determining salary outcomes  

---

### 🛠️ Most Common Skills in the Market  
![Top skills in the market](0_Resources/Images/2_Project_Analysis_Chart3.png)
- SQL and Python consistently appear across the majority of data roles  
- Cloud technologies (such as AWS and Azure) are increasingly important  
- Core technical tools dominate higher-paying job listings  

---

### 💰 Skills Linked to Higher Pay  
![Skills linked to higher pay](0_Resources/Images/2_Project_Analysis_Chart4.png)
- High-paying roles are strongly associated with programming and cloud-based skills  
- More general tools and office-based skills tend to appear in lower salary bands  
- The combination of multiple technical skills increases earning potential  

---

## 🔎 Analytical Approach  

To conduct this analysis, I:
- Cleaned and structured job posting data  
- Grouped data by job title, country, and skills  
- Used Pivot Tables to summarise salary distributions  
- Compared medians across categories  
- Identified trends between skills and compensation
